# Spotiva

**Spotiva** to mobilna aplikacja, która pozwala w prosty i szybki sposób dokumentować swoje ulubione miejsca i dzielić się nimi ze znajomymi. Tworzysz zdjęcia, dodajesz ich lokalizację i udostępniasz je w mapowej galerii — wszystko w jednym miejscu! To połączenie aparatu, mapy i społeczności w jednej aplikacji.

## Funkcjonalności

- **Spot** – zrób zdjęcie miejsca.  
- **Share** – udostępniaj komu chcesz.  
- **Shine** – pozwól innym odkryć Twoje ulubione miejsca i zainspiruj ich swoimi chwilami.
